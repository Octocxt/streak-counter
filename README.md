# 🔥 Streak Counter

A clean, gamified streak tracker designed to help you stay consistent and disciplined — one day at a time.

## 🌐 Live Demo

👉 https://octocxt.github.io/streak-counter/

---

## ✨ Features

* ⚔️ Daily check-in system (one update per day)
* 💀 Fail & reset system with confirmation modal
* 🏆 Rank progression system (Mortal → Eternal)
* 📊 Progress bar toward next rank
* 🎯 Best streak tracking
* ✏️ Manually set streak option
* 🎨 Dynamic UI (colors, glow, animations based on progress)
* 💾 Data persistence using `localStorage`

---

## 🧠 How It Works

* Click **"Succeeded"** once per day to increase streak
* Click **"Failed"** to reset streak
* Progress unlocks new **titles and ranks**
* Data is saved locally in your browser
